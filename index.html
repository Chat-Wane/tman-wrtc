    tman-wrtc

    <i>Keywords: T-Man, configurable network topology, WebRTC</i>

    This project aims to provide an implementation of the T-Man [1] using WebRTC. T-Man is a protocol that builds an overlay network based on descriptors. Created by the developer, and exploiting the descriptors, a fitness function allows choosing the right neighbourhood.

    Installation

    $ npm install tman-wrtc

    References

    [1] M. Jelasity, A. Montresor, and O. Babaoglu. T-man: Gossip-based fast overlay topology construction<i> Comput. Netw., vol. 53, no. 13, pp. 2321-2339, 2009.</i>
