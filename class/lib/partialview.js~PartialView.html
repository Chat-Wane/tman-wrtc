    PartialView

    Extends:

    Map → PartialView

    Structure containing the neighborhood of a peer. Each neigbhor entry comprises a list of ages and a descriptor. Map of {idPeer => {ages:[age_1, age_2.. age_k], descriptor: {x: exampleX} }} where age_1 <= age_2 <= .. <= age_k.
