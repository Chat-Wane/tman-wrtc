    IRPS

    Extends:

    EventEmitter → IRPS

    An interface providing easy-to-use functions on top of Spray. Once the protocol registered to Spray, it gets this interface. It can send messages using irps.emit('eventName', neighborId, args) and the neighbor can catch them using irps.on('eventName', args).
